# 🌱 FARM SENSE
### 실시간 감지 센서와 원격·자동 제어를 활용한 IoT 기반 스마트 농작물 관리 시스템

## 📌 프로젝트 개요
**FARM SENSE**는 주말농장 및 소규모 도시농업 환경에서 발생하는 관리 어려움을 해결하기 위해  
IoT 센서, AWS IoT Core, 모바일 애플리케이션을 활용하여  
**실시간 모니터링 · 자동 제어 · 원격 제어**를 제공하는 스마트 농작물 관리 시스템입니다.

도시민이 농장을 직접 방문하지 않아도 작물의 생육 환경을 안정적으로 유지할 수 있도록 설계되었습니다. :contentReference[oaicite:0]{index=0}

| 이름 | 역할 |

## 🎯 기획 배경
- 도시농업 참여 인구의 지속적인 증가
- 주말농장의 **접근성 문제** 및 **상시 관리의 어려움**
- 기후 변화로 인한 작물 생육 불안정
- 농업 경험이 부족한 사용자도 쉽게 활용 가능한 자동화 시스템의 필요성

---

## ⚙️ 주요 기능
### 1️⃣ 실시간 환경 모니터링
- 온도, 습도, 조도, 공기질(CO₂) 센서를 통한 데이터 수집
- OLED 디스플레이 및 모바일 앱에서 실시간 확인

### 2️⃣ 자동 제어 시스템
- 온도 기준 미달 시 히터(LED) 자동 작동
- 토양 습도 부족 시 워터 펌프 자동 작동
- 조도 부족 시 보조 조명 자동 점등
- CO₂ 농도 및 온·습도 초과 시 팬 자동 작동

### 3️⃣ 원격 수동 제어
- 모바일 애플리케이션을 통한 히터, 펌프, 조명, 팬 수동 제어
- 자동 모드 / 수동 모드 전환 기능 제공

### 4️⃣ 작물별 맞춤 환경 설정
- 작물별 적정 온도·습도 가이드라인 제공
- 사용자 설정 값 기반 자동 제어 로직 적용

### 5️⃣ 데이터 저장 및 활용
- AWS DynamoDB에 사용자 설정 및 환경 데이터 저장
- 사용자 데이터 기반 작물별 환경 가이드라인 제공
